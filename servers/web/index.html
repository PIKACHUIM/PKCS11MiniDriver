🔐 OpenCert Platform

开源的 CA + 智能卡 + X.509/GPG/SSH 证书管理平台，提供完整的证书生命周期管理、虚拟智能卡驱动和 PKCS#11 标准兼容接口。

API 状态 GitHub
🏛️

CA 证书颁发

完整的 CA 管理功能，支持根 CA 创建、中间 CA 签发、证书模板管理、CRL/OCSP 吊销服务和 ACME 自动化签发。

💳

虚拟智能卡

三种安全等级的虚拟智能卡（TPM 高安全/本地中安全/密码低安全），兼容 PKCS#11 标准，可被系统和应用直接调用。

🔑

多类型凭据

统一管理 X.509 证书、SSH 密钥、GPG 证书、TOTP/FIDO 认证、登录信息、安全笔记等多种安全凭据。

☁️

云端同步

证书和密钥可安全存储在云端智能卡中，支持离线缓存、自动重连和端到端加密的私钥备份与恢复。

🛡️

安全架构

AES-256-GCM 三层加密、Argon2id 密钥派生、TPM 硬件保护、JWT Token 轮换、RBAC 权限控制。

💰

支付与订单

灵活的支付插件系统，支持多种支付渠道，完整的充值、订单、余额管理和退款审批工作流。

系统架构

三层架构设计，驱动层通过 IPC 与管理端通信，管理端通过 REST API 与云端平台交互。

OpenCert Driver  (pkcs11-mock.dll/so)    — PKCS#11 标准驱动
       ↕  IPC (Named Pipe / Unix Socket)
OpenCert Manager (client-card + webpage) — 本地管理端
       ↕  REST API (HTTPS)
OpenCert Platform (server-card)          — 云端平台服务
